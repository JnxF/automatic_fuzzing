# Automatic fuzzing

Based on the [Creating custom ROS 2 msg and srv files](https://index.ros.org/doc/ros2/Tutorials/Custom-ROS2-Interfaces/) tutorial, we aim to automatize fuzzing on ROS 2.

## Installation

Have Docker installed.

## Usage

```
source start.sh
colcon build
```
